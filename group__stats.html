Detailed Description

Speed-optimized statistics functions.

These functions calculate basic sample statistics. Unlike the equivalent GSL functions, they work on any container and do not require the overhead of converting to C-array format. Use them by including stats.tmp.h. Include stats_except.h to catch exceptions thrown by these functions.

Tests whether a range is sorted.

This function emulates std::is_sorted() for platforms without access to either a C++11-compliant compiler or Boost 1.50 or later.

Template Parameters
ConstForwardIteratorThe iterator type for the container to be tested. Must be a forward iterator.
Parameters
[in]first,lastForward iterators to the range to test. The range is all elements in the interval [first, last).
Returns
True if [first, last) is sorted in ascending order. A range containing less than two elements is always sorted.
Performance
O(D), where D = std::distance(first, last).
Exception Guarantee
The range [first, last) is unchanged in the event of an exception if ConstForwardIterator provides at least the basic guarantee. Does not throw exceptions unless ConstForwardIterator throws.
Test:

Empty list. Expected behavior: true.

List of size 1. Expected behavior: true.

List of size 2, sorted asc. Expected behavior: true.

List of size 2, sorted desc. Expected behavior: false.

List of size 10, list[i] = i^2. Expected behavior: true.

List of size 10, list[i] = (i-5)^2. Expected behavior: false.

List of size 10, random elements. Expected behavior: false.

Finds the mean of the values in a generic container object.

The container class is accessed using first and last iterators, and the mean is computed over the interval [first, last).

Template Parameters
ConstInputIteratorThe iterator type for the container over which the mean is to be calculated. Must be readable and support forward traversal.
Parameters
[in]firstInput iterator marking the first element in the container.
[in]lastInput iterator marking the position after the last element in the container.
Returns
The arithmetic mean of the elements between first, inclusive, and last, exclusive. The return type is that of the elements pointed to by the first and last iterators.
Precondition
[first, last) is a valid range
There is at least one element in the range [first, last)
No value in [first, last) is NaN
Postcondition
The return value is not NaN
Performance
O(D), where D = std::distance(first, last).
Exceptions
kpfutils::except::NotEnoughDataThrown if there are not enough elements to define a mean.
Exception Guarantee
The range [first, last) is unchanged in the event of an exception.
Test:

List of ints, length 0. Expected behavior: throw invalid_argument.

List of ints, length 1. Expected behavior: return list[0]

List of ints, length 100, randomly generated. Expected behavior: agrees with gsl_stats_mean in 10 out of 10 trials.

List of doubles, length 100, randomly generated. Expected behavior: agrees with gsl_stats_mean to within 1e-10 in 10 out of 10 trials.

Vector of doubles, length 100, randomly generated. Expected behavior: agrees with gsl_stats_mean to within 1e-10 in 10 out of 10 trials.

Array of doubles, length 100, randomly generated. Expected behavior: agrees with gsl_stats_mean to within 1e-10 in 10 out of 10 trials.

Finds the (uninterpolated) quantile of the values in a generic container object.

The container class is accessed using first and last iterators, and the quantile is computed over the interval [first, last). Data is assumed unsorted.

Template Parameters
ConstRandomAccessIteratorThe iterator type for the container over which the variance is to be calculated. Must be readable and support random access
Parameters
[in]firstInput iterator marking the first element in the container.
[in]lastInput iterator marking the position after the last element in the container.
[in]quantileThe percentile to recover.
Returns
The largest value whose quantile is less than or equal to quantile.
Precondition
[first, last) is a valid range
There are at least two elements in the range [first, last)
0 <= quantile <= 1
No value in [first, last) is NaN
Postcondition
The return value is not NaN
Performance
O(D log D), where D = std::distance(first, last).
Exceptions
std::invalid_argumentThrown if quantile is not in the interval [0, 1].
kpfutils::except::NotEnoughDataThrown if there are not enough elements to define a quantile.
Exception Guarantee
The range [first, last) is unchanged in the event of an exception.
