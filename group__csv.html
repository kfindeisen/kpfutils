Prints a file containing a histogram.

Parameters
[in]fileNamethe name of a file to be written to
[in]binEdgesa vector of starting and ending bin edges
[in]valuesa vector of bin heights
Precondition
binEdges.size() = values.size()+1
for all i, values[i] is the number of items between binEdges[i] and binEdges[i+1]
Postcondition
Produces a text file containing two space-delimited columns containing the bin edges and values, respectively. If the file already exists, it is replaced.
Exceptions
std::invalid_argumentThrown if binEdges.size() ≠ values.size()+1
kpfutils::except::FileIoThrown if any file operation fails.
Exception Guarantee
Program is in a consistent state in the event of an exception.

void kpfutils::readTable ( const string &  fileName,
const string &  format,
vector< double > &  col1,
vector< double > &  col2 
)

Reads a file containing two columns of data.

Parameters
[in]fileNamethe name of a file to be read. The file is assumed to be formatted as a 2xN table, with each row in the format given by format. The file may also contain comment lines preceded by '#'.
[in]formata scanf-style formatting string representing a single row of the table in hInput
[out]col1,col2vectors containing the columns of the table
Precondition
scanf(format, ...) reads exactly two floating-point values
Postcondition
col1 and col2 contain the data contained in the two selected columns of fileName
col1.size() = col2.size()
Exceptions
std::bad_allocThrown if there is not enough memory to store the data.
kpfutils::except::FileIoThrown if any file operation fails.
Exception Guarantee
Program is in a consistent state in the event of an exception.
