void errorFilter ( double  errMax,
vector< double > &  times,
vector< double > &  data,
vector< double > &  errs 
)

Removes all (time, data, error) triplets exceeding an error threshold.

Parameters
[in]errMaxthe maximum error to tolerate in a data point
[in,out]timesa vector containing the times of each observation
[in,out]dataa vector containing the measurement (typically flux or magnitude) observed at each time
[in,out]errsa vector containing the error on each measurement
Invariant
times.size() = data.size() = errs.size()
for all i, data[i] ± errs[i] is the measurement taken at times[i]
Postcondition
all elements of errs exceeding errMax are removed, along with all corresponding elements of times and data
Exceptions
std::bad_allocThrown if there is not enough memory to modify the data.
Exception Guarantee
The function arguments are unchanged in the event of an exception.
