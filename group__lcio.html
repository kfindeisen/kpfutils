Detailed Description

These functions read and write light curves to disk.

Include lcio.h to use these functions

Include lcexcept.h to catch related exceptions.

void kpfutils::filterLightCurve ( double  date1,
double  date2,
DoubleVec &  times,
DoubleVec &  arr1,
DoubleVec &  arr2 
)

Filters a set of vectors to include only dates between date1 and date2.

Parameters
[in]date1the smallest Julian date to be allowed in times
[in]date2the largest Julian date to be allowed in times
[in,out]timesa vector of times to be trimmed to [date1, date2]
[in,out]arr1,arr2vectors of measurements corresponding to times, to be trimmed in parallel
Precondition
times is sorted in ascending order
Postcondition
times is updated to contain only items greater than or equal to date1 and less than or equal to date2. Any entries corresponding to deleted entries of times are deleted from arr1 and arr2.
Exceptions
kpfutils::except::NoValidTimesThrown if date2 < date1 or if all values in times are outside [date1, date2].
std::bad_allocThrown if there is not enough memory to reprocess the vectors
Exception Guarantee
The function arguments are unchanged in the event of an exception.

void kpfutils::readWg2LightCurve ( const string &  fileName,
double  errMax,
DoubleVec &  timeVec,
DoubleVec &  dataVec,
DoubleVec &  errVec 
)

Reads a file containing a list of obsids, Julian days, measurements, errors, and limits.

Parameters
[in]fileNamethe name of a file to be read. The file is assumed to be formatted as a 5xN space-delimited table, with the first column a running index, the second a floating point Julian date, the third a floating point measurement, the fourth a floating point error, and the fifth a detection limit in the same units as the measurement. The file may also contain comment lines preceded by '#'.
[in]errMaxthe maximum error to tolerate in a data point. Any points with an error exceeding errMax are ignored.
[out]timeVeca vector containing the times of each observation
[out]dataVeca vector containing the measurement (typically flux or magnitude) observed at each time
[out]errVeca vector containing the error on each measurement
Postcondition
timeVec is sorted in ascending order
timeVec.size() = dataVec.size() = errVec.size()
for all i, dataVec[i] ± errVec[i] is the measurement taken at timeVec[i]
for all i, errVec[i] ≤ errMax
Exceptions
std::bad_allocThrown if there is not enough memory to store the data.
kpfutils::except::FileIoThrown if any file operation fails.
Exception Guarantee
The function arguments are unchanged in the event of an exception.
Bug:
Current implementation ignores limits.
