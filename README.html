kpfutils Read-Me

The contents of this directory, and the the kpfutils namespace, are not a single library. Rather, they are a set of common tools and conventions I use for writing astronomical applications. They are needed to compile other C++ programs I have written, but are not intended for standalone use.

More information, including installation instructions, can be found in the kpfutils Library Reference Manual, provided with the source distribution in HTML and PDF format.

For questions or feedback, please contact Krzysztof Findeisen.
