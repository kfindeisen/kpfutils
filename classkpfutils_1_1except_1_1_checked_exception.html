Detailed Description

This is the base class for exceptions thrown by Lightcurve MC that are foreseeable and can, in principle, be handled at run time.

This class models the "checked exception" concept from Java. It is therefore a conceptual sibling to std::runtime_error, which is similar to Java's errors, and std::logic_error, which is similar to Java's runtime exceptions. As such, CheckedException has an analogous interface to std::runtime_error and std::logic_error.
